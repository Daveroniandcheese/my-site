This is a placeholder post so the build pipeline has something to render. Drop
a new `.mdx` file into `app/blog/posts/` to publish a post — the file name
becomes the URL slug, and the frontmatter at the top populates the title,
date, and summary.

## Markdown works

You can write **bold**, _italic_, and [links](https://example.com) like
normal. Code blocks get syntax highlighting:

```ts
function greet(name: string) {
  return `Hello, ${name}`
}
```
